In **Internet Explorer** you might get a yellow banner which says "Searching for photos, please wait" which goes on for ever. If so, you probably need to turn on Compatibility View for this website:

  Go to Tools | Compatibility View settings  
  Add the website "uk.com" to the list.  
  Try viewing photos again.

The Photos page no longer goes straight to a random race, but waits for you to select a race.

Uploading photos gives you two options:  
  Link to a news item - in which case the photos you upload will appear with the news item you select.  
  Link to a race - in which case they will appear in a group via the Photos tab.
